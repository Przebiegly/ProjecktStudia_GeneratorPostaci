# Dokumentacja Projektu: Generator Postaci Warhammer

## 1. Instrukcja Uruchomienia

```bash
# 1. Pobierz aplikacje 
# 2. Upewnij się, że masz zainstalowany Python.
# 3. Zainstaluj bibliotekę PyQt6:
pip install PyQt6
# 4. Uruchom aplikację:
python main.py
```

## 2. Instrukcja Działania (ze zdjęciami)

*   **Okno Kreatora Postaci**

    ![Obraz Okna Początkowego](photos/First_window.png)

Powyższy zrzut ekranu przedstawia pierwsze okno, które zobaczysz po uruchomieniu aplikacji. To tutaj rozpoczyna się przygoda – tworzysz swoją postać. Masz do wyboru trzy główne ścieżki:

    1.  **Tworzenie Ręczne:** Możesz samodzielnie przejść przez cały proces, wybierając rasę, płeć, imię, profesję i inne szczegóły. Formularz jest interaktywny – po wybraniu rasy i profesji, podgląd cech, umiejętności i zdolności na dole okna automatycznie się zaktualizuje. To idealna opcja, jeśli masz już w głowie konkretny koncept na swojego bohatera. Po uzupełnieniu wszystkich pól, kliknij przycisk **"Stwórz Postać"**.

    2.  **Wczytanie Postaci:** Jeżeli masz już zapisaną postać z poprzedniej sesji, nie musisz tworzyć nowej. Użyj przycisku **"Wczytaj Postać"**, aby otworzyć okno z listą swoich zapisów i kontynuować grę.

    3.  **Generowanie w Pełni Losowe:** Jeśli chcesz szybko wskoczyć do gry lub szukasz inspiracji, kliknij przycisk **"Wygeneruj w Pełni Losową Postać"**. Aplikacja zrobi wszystko za Ciebie – od rasy i imienia, po cechy i startowy ekwipunek, tworząc kompletną, gotową do gry postać jednym kliknięciem.
